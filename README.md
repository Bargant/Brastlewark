# Brastlewark

## Introduction

Brastlewark census app is a applciation built in React that can search gnomes from the Brastlwark city census. It is built using React.

## Packages used

```
bootstrap
react-input-range
react-paginate
```

# Installation

You can run it locally using:
'''
$ npm install
$ npm run start
'''
